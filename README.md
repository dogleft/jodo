## Picking a Jo
- Length: 4 Shaku, 2 Sun, 1 Bu (~127.6 cm) (~50.25")
- Diameter: 8 Bu (~2.4 cm) (~15/16")
- Unvarnished
- [Cool length converter](https://www.kampaibudokai.org/Script.htm)
